# AI-Portfolio-Lab

A massive collection of **76 distinct, highly creative, and interactive 3D web concepts**. This repository serves as an experimental laboratory integrating Next-Gen User Interfaces and real-time AI generation via the Google Gemini API. All creations are the work of **Ömercan Sabun**.

*Note: Since this is a static showcase without a backend server, the site runs completely in your browser. To trigger the AI animations and responses, you will be prompted to enter your own Gemini API Key locally. Your key is stored solely in your browser's `localStorage` and never leaves your device.*

## 🚀 Features & Aesthetics

This collection explores a wide spectrum of design languages and rendering techniques, including but not limited to:
- **Interactive 3D Visuals:** Powered by `Three.js`, featuring fluid dynamics, mechanical parts, particles, physics simulations, and procedural geometry.
- **Dynamic CSS Typography & Layouts:** Neobrutalism, Bento Grids, Glassmorphism, Retro OS (Windows 95, Mac OS 9), and Terminal CLI aesthetics.
- **AI Streaming Integration:** Real-time markdown parsing with `Marked.js` and dynamic flowchart generation with `Mermaid.js`.
- **Intelligent Feedback Loops:** The 3D environments react physically (speed, color, shape, audio) to the active state of the AI generation.
- **Enterprise Virtualization Dashboard:** The master index is a fully immersive 3D grid with hover previews and language localization (English & Turkish).

## 📁 Repository Structure

Each numbered folder (e.g., `01_futuristic_scifi`, `26_gemini_prompt`, `76_megastructure`) represents a completely standalone frontend concept with its own unique HTML, CSS, and JS logic.

## 🛠️ Technologies Used

- **HTML5 & CSS3** (Vanilla, No frameworks for pure flexibility)
- **JavaScript** (ES6+)
- **Three.js** (WebGL 3D Rendering)
- **Anime.js & GSAP** (Advanced DOM animations)
- **Google Gemini API** (LLM Integration)
- **Marked.js** (Markdown Parsing)
- **Mermaid.js** (Diagram Generation)
